# Jaipur IOC 2009 Public-Incident Summary Stub

The Jaipur IOC tank-farm fire is represented in Verge as a reconstructed replay
stub for eval and memory retrieval. The useful safety pattern is hydrocarbon
release, ignition exposure, delayed isolation, and operational handover under
uncertain field conditions.

Relevant pattern: tank-farm gas drift, concurrent field activity, and unclear
permit/zone boundaries can combine into a high-consequence event before a
single dashboard explains the convergence.
